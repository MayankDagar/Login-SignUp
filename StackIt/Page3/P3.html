How to center a div vertically and horizontally in CSS?

I've tried various methods but none seem to work consistently across browsers. What's the modern way to center a div both vertically and horizontally?

css html flexbox
asked 2 hours ago by user123

2 Answers

8

The modern way to center a div both vertically and horizontally is to use CSS Flexbox: 

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

This will work in all modern browsers.

3

You can also use CSS Grid for centering: 

.container {
    display: grid;
    place-items: center;
    height: 100vh;
}

This is even more concise than the Flexbox solution.
